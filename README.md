1. 1题目描述

哈夫曼编码/译码系统，对字符串进行编码/译码

本系统实现了哈夫曼编码/译码的功能，具有对字符串进行编码/译码，并且把结果保存到文件，读取文件进行编码/译码等功能；

提示：可自行编码/译码内容提前保存到code.txt/encode.txt文件中

要求：可选择是否将结果保存到文件中；

1. 4概要设计

程序由数据结构和算法两个要素构成。所谓数据结构，即程序的操作对象；所谓算法，即解决问题的方法和步骤，解决一个具体问题的算法，也称为操作。

##### 4.1 数据结构

哈夫曼编码/译码器中，涉及到的数据为字符，字符的权值，还有字符的编码值保存，在实现中采用结构体数组，定义如下：

typedef struct HuffmanNode //定义哈夫曼树各结点

{

        char c;//字符

 int weight;  //权值

 int parent;  //父节点

 int lchild, rchild;  //左右孩子编号

}HuffmanNode;

typedef struct //存放哈夫曼树根节点对应的编码值

{

 char cd[N];                 //存放哈夫曼码

 int start;                  //从start开始读cd中的哈夫曼码

}HCode;

此外，还需定义全局变量：

char textTemp[1000];   //暂时存放编码或者译码输出内容，用来保存到文件

##### 4.2 主要操作

系统中涉及的主要操作如下：

1. （1）void CreateHT(HuffmanNode ht[], int n)

创建哈夫曼树；

1. （2）void CreateHCode(HuffmanNode ht[], HCode hcd[], int n)

根据哈夫曼树生成哈夫曼编码表

1. （3）void deHcode(HuffmanNode ht[], int n,char\* temp)

把电文（01）翻译成字符串；

1. （4）void enHCode(char\* string, HuffmanNode ht[], HCode hcd[], int n)

根据编码表对字符串进行编码成01串;

1. （5）void saveText()

将全局变量数组char textTemp[1000]的内容保存到文件中

1. （6）void DispHCode(HuffmanNode ht[], HCode hcd[], int n)

输出编码表内容

1. （7）int readHC( HuffmanNode ht[])

读取文件in.txt，根据给出的字符及权值生成哈夫曼树，并返回不相同的字符的个数；

1. （8）int readText(HuffmanNode ht[])

读取文件test.txt，根据根据文章内容生成哈夫曼树，并返回不相同的字符的个数；

1. （9）int buitlHu(char str[], HuffmanNode ht[])

根据char str[]字符串数组内容构建哈夫曼树;

1. （10）void textEnCode(char\* textNname, HuffmanNode ht[], HCode hcd[], int n)

把文件中的字符串根据编码表编码为01串

1. （11）void textDeCode(char\* textNname, HuffmanNode ht[], int n)

对文件中的01串进行译码
